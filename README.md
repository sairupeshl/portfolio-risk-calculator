# Portfolio Risk Analytics Calculator
*Tech:* Python (Pandas, NumPy), Yahoo Finance API

- Engineered a portfolio risk engine using Python, integrating the Yahoo Finance API to ingest and preprocess historical multi-asset price data.
- Calculated Parametric Value-at-Risk (VaR) at a 95\% confidence level by constructing the covariance matrix to evaluate overall portfolio volatility.
- Executed historical backtesting on daily portfolio returns to quantify tail-risk exceptions, comparing empirical breach rates against the expected 5\% statistical threshold to validate the VaR model's predictive accuracy.
